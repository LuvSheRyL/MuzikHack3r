# MuzikHack3r

这是一份关于音乐的报告，涵盖音乐资源工具箱，开源项目，音乐史，学习路线等。

在项目建立初期，原计划以Loop音乐为主，命名以Tupac惯用的缩写和Eminem的变体方式致敬。

先更再说吧。更完再谈致敬之类的。
